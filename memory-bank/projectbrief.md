# Project Brief: SNNAKE Barbershop Booking Funnel

## Overview
A high-converting single page application (SPA) with a booking funnel for SNNAKE Barbershop in Царево. The site is designed as a conversion-focused funnel to guide visitors to book an appointment within 30 seconds.

## Core Requirements
- Premium, masculine, dark mode design (black/dark grey + gold accents)
- Multi-step booking funnel (Service → Barber → Date/Time → Details → Confirmation)
- Admin dashboard for appointment management
- Mobile-first, responsive design
- Micro-animations for premium feel

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS 3.4 with custom dark/gold theme
- Framer Motion for animations
- React Hook Form + Zod for form validation
- Lucide React for icons
- Supabase-compatible backend structure

## Primary Goal
Get users to click "Book Now" and complete their reservation within 30 seconds.
